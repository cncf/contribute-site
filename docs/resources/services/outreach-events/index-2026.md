## KubeCon + CloudNativeCon

KubeCon + CloudNativeCon is the primary global conference series organized by CNCF. These events bring together maintainers, contributors, and end users of cloud native technologies.

CNCF projects and contributors may participate through:
- Maintainer Track sessions
- Lightning Talks
- Project Pavilion or kiosk participation
- CNCF-hosted co-located events
- Contributor and maintainer meetups

KubeCon + CloudNativeCon events are held multiple times per year in different regions.

More information:  
https://www.cncf.io/events/

## Kubernetes Community Days (KCDs)

Kubernetes Community Days (KCDs) are **community-organized, CNCF-supported** events focused on local Kubernetes and cloud native communities.

KCDs provide opportunities to:
- Deliver first-time or project-focused talks
- Build and support local contributor communities
- Share experiences using CNCF projects
- Organize or volunteer at community-led events

KCDs take place throughout 2026 in multiple locations worldwide.

More information:  
https://community.cncf.io/kubernetes-community-days/

## CNCF community events and meetups

CNCF supports a global network of **local chapters, special interest groups, and virtual communities** that host events year-round.

These events may include:
- Technical talks and workshops
- Project demonstrations and updates
- Contributor onboarding sessions
- Career and community discussions

Community events occur on an ongoing basis throughout 2026.

More information:  
https://community.cncf.io/events/

## Calls for proposals and submissions

Many CNCF-supported events accept proposals for talks, panels, or project participation.

Common submission opportunities include:
- KubeCon + CloudNativeCon calls for proposals
- Maintainer Summit calls for proposals
- Kubernetes Community Days talk submissions
- CNCF community meetup talk submissions

Submission timelines and requirements vary by event.

More information:  
https://contribute.cncf.io/resources/services/outreach-events/

## Contributor programs related to events

The following CNCF-supported programs often align with outreach and event participation, including presentations or demos at CNCF events.

### LFX Mentorship

LFX Mentorship provides structured mentorship opportunities for contributors working on CNCF projects. Programs run multiple times per year.

More information:  
https://lfx.linuxfoundation.org/tools/mentorship/

### Google Summer of Code (GSoC)

CNCF participates in Google Summer of Code as a mentoring organization. Contributors work on CNCF projects and may present their work publicly.

More information:  
https://summerofcode.withgoogle.com/

## Notes for CNCF projects

- Projects should coordinate outreach and event participation with CNCF Event Services where applicable.
- Outreach planning should begin early to allow sufficient review and scheduling time.
- Past events are intentionally excluded from this page.
