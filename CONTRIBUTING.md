# Contributing

Thanks for your interest in contributing to this website!

> [!IMPORTANT]
>
> All contributors are bound by the [CNCF Code of Conduct][], please read it
carefully.

## Ways to contribute

### Report a problem

If you see a technical issue with the website or something that doesn't look right, search [existing issues][] to see if it has been reported already. If not, [create a new issue][] describing the problem. Provide all steps necessary to reproduce the problem and screenshots when applicable.

### Fix a issue

Find [an issue][existing issues] you want to fix and submit a PR for approval.

## Setting up

Read about [how to set up for local development][setup-locally] in the README.

## Need help?

To connect with our community and ask questions, join the [CNCF Slack
workspace](https://slack.cncf.io/).
